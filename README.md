📊 Linear Regression Project

This project implements Linear Regression to predict values and understand relationships between data using Python.

🚀 Features

Linear Regression from scratch

Model training using gradient descent

Data visualization using graphs

🛠️ Tech Stack

Python

NumPy

Matplotlib

▶️ Run
git clone https://github.com/SANJEEVKUMAR/linear-regression.git
cd linear-regression
pip install -r requirements.txt
python train.py
📊 Output

Shows a regression line fitting the data points.
